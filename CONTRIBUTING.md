# Contributing

Thanks for your interest in contributing!

## How to contribute
- Open an issue to describe the problem or proposal.
- For code changes, create a feature branch and submit a pull request.
- Keep changes focused and include tests when applicable.

## Development
- Use the Go version specified in `go.mod`.
- Run `go test ./...` before submitting changes.

## Code style
- Keep code readable and idiomatic.
- Add or update English doc comments for exported symbols.
